Generar un generador

Un generador es, en su núcleo, un módulo Node.js.

Instalar globalmente Yeoman

npm install -g yo

Instalar globalmente Gulp

npm install -g gulp

Instalar globalmente generator-generator

npm install -g generator-generator

Generar un generator

yo generator
